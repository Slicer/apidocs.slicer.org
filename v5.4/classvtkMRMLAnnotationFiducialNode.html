Detailed Description

MRML node to represent a fiducial in the Annotations module - deprecated.

Annotation nodes contains control points, internally represented as vtkPolyData. A Annotation node contains many control points and forms the smallest logical unit of tractography that MRML will manage/read/write. Each control point has accompanying data. Visualization parameters for these nodes are controlled by the vtkMRMLAnnotationPointDisplayNode class.
