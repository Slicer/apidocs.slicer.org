◆ SetFastPick3D()

virtual void vtkMRMLCrosshairNode::SetFastPick3D ( bool )
virtual

Experimental feature to choose between fast or accurate position computation in 3D views. If fast option is chosen then Z-buffer in the renderer may be used for determining position along the view line in 3D views, which is very fast but may not give optimal results because semi-transparent objects, such as semi-transparent models or volume-rendered images are ignored. This property is only for evaluation of this feature and the value is not stored persistently in the scene file.
