Detailed Description

Legacy subject hierarchy node to allow loading older MRML scenes.

Older MRML scenes contain per-item subject hierarchy nodes, which are incompatible with the new single-node subject hierarchy design. Those nodes (identified by the lack of vtkMRMLSubjectHierarchyNode::SUBJECTHIERARCHY_VERSION_ATTRIBUTE_NAME attribute) are converted to this node type.

Notes:

  • Only those methods are kept in this node from the old one that may be necessary to extract or use information loaded into this not and are not propagated into the new hierarchy (such as UIDs).
  • Nested association support is also removed (because it never became completely stable), so if a scene is loaded where a data node is associated to the subject hierarchy node using nested association, then that node will not be properly represented in the hierarchy.
