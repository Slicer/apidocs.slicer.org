Detailed Description

MRML node for representing diffusion weighted MRI volume.

Diffusion Weighted Volume nodes describe data sets that encode diffusion weighted images. These images are the basis for computing the diffusion tensor. The node is a container for the necessary information to interpret DW images:

  1. Gradient information.
  2. B value for each gradient.
  3. Measurement frame that relates the coordinate system where the gradients are given to RAS.
