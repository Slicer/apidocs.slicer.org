Detailed Description

MRML node for representing a sequence of MRML nodes.

This node contains a sequence of nodes (data nodes).

The data nodes can be referred to by an index, for example: IndexName: time IndexUnit: s IndexNumeric: YES IndexValue: 3.4567

If an index is numeric then it is sorted differently and equality determined using a numerical tolerance instead of exact string matching.

Class name of data nodes stored in the sequence is set into the DataNodeClassName node attribute, which may be used for attribute-based filters (for example, to show only certain type of sequence node in a node selector).
