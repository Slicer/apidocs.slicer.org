Detailed Description

Combobox of display node view nodes. Observe the view nodes of a display node and mark them as checked in the scene view node list. View nodes can be vtkMRMLViewNode for the 3D view or vtkMRMLSliceNode for the 2d case, or vtkMRMLPlotChartViewNodes for charts.
