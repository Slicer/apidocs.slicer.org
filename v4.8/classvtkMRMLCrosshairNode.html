◆ SetCursorPositionRAS()

void vtkMRMLCrosshairNode::SetCursorPositionRAS ( double  ras[3])

Set cursor position in 3D. This should be called whenever the cursor is moved by using a 3D positioning device (that does not have an associated slice viewer). GetCursorPositionXYZ returns with NULL if SetCursorPositionRAS is used for setting the cursor position, so if slice position is available then SetCursorPositionXYZ method should be used instead (it sets both XYZ and RAS position, too).
