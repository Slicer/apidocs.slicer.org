Detailed Description

MRML node to represent a curve markup Curve Markups nodes contain N control points. Visualization parameters are set in the vtkMRMLMarkupsDisplayNode class.

Markups is intended to be used for manual marking/editing of point positions.

Coordinate systems used:

  • Local: Current node's coordinate system where the position of the control and curve points are defined. Local coordinates can be converted to world by concatenating all parent transforms on the current node.
  • Surface: Model node's coordinate system where the polydata used for ShortestDistanceOnSurface pathfinding is defined. Surface coordinates can be converted to world by concatenating all parent transforms on the surface node.
  • World: Patient coordinate system (RAS)

Markups measurement pipeline:
