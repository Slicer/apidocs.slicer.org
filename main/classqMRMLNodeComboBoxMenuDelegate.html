Detailed Description

Abstract Item delegate to show unselectable items highlighted when the mouse is over them to give a feeling of selectable item.

This is used in qMRMLNodeComboBox for the extra items: e.g. "Add/Remove/Edit node" actions The highlight color is slightly more transparent than the style palette highlight color to indicates that the item is not really selectable but correspond to an action.

We reimplement the code from Qt "QComboMenuDelegate" used in the class QComboBox.
