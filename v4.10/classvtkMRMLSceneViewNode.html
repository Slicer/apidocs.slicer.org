◆ RestoreScene()

void vtkMRMLSceneViewNode::RestoreScene ( bool  removeNodes = true)

Restore content of the scene from the node. If removeNodes is true (default), remove nodes from the main Slicer scene that do no appear in the scene view. If it is false, and nodes are found that will be deleted, don't remove them, print a warning, set the scene error code to 1, save the warning to the scene error message, and return.
